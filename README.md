# Website Creatie

Een project voor het bouwen van een website.

## Projectstructuur

```
website-creatie/
├── index.html       # Hoofdpagina
├── css/             # Stijlbestanden
├── js/              # JavaScript-bestanden
└── assets/          # Afbeeldingen en andere media
```

## Aan de slag

Open `index.html` in je browser om de website te bekijken.
